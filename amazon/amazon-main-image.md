# Amazon Main Image — 亚马逊主图

## Scene Type
Amazon main image / 亚马逊主图（白底纯产品图）

> **Note:** Amazon's main listing image must have a pure white background with the product only — no text, logos, or props. This prompt follows that requirement.
>
> **注意：** 亚马逊主图要求纯白背景，仅展示产品本身，不得有文字、Logo或道具。本提示词符合该规范。

---

## English

Extract the carpet from the reference image and produce a clean, top-down overhead product shot. Precisely reproduce the carpet's original pattern, colors, material, and texture. Center the carpet on a pure white background, fully visible with no cropping or distortion — correctly accounting for the perspective shift from the reference to ensure accurate size proportions. 1:1 square format, 8K resolution, razor-sharp focus, minimal soft shadow, professional white-background product photography. No text, no props, no watermark.

---

## 中文

提取参考图中的地毯，生成干净的正俯视商品主图。精确还原地毯的图案、色彩、材质与纹理，居中铺于纯白背景上，完整展示无裁切，正确处理参考图透视带来的形变，确保转换后尺寸比例准确。1:1正方形比例，8K分辨率，焦点锐利，极简柔和阴影，专业白底产品摄影。无文字、无道具、无水印。
